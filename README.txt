# 🌊 スサノオOS  
### 神話の心と技術の力で、OSを再創造する。

日本神話の世界観と現代の技術を融合した、  
国産の実験的オペレーティングシステム・プロジェクト。

---

## 🌿 コンセプト

スサノオOSは、日本神話「スサノオ」の名を冠した  
**人と機械の新しい関係（MMI：Man Machine Interface）** を探る国産OSです。

「和の思想」と「現代技術」の融合によって、  
日本的な感性と文化をベースにした新しいOS体験を目指します。

---

## ⚙ 技術的特徴

- 🧭 **EDK2＋UEFI**：ブートローダから独自構築  
- 🧠 **軽量カーネル**：Cで実装、テキスト描画からGUIまで自前で展開  
- 🪄 **神話×アプリ構成**：  
  - ニニギ → ファイルマネージャ  
  - ツクヨミ → カレンダー／TODO  
  - アマテラス → ブラウザ  
- 🌱 **育てるOS**：開発者・ユーザーが共に拡張できる“余白”ある設計
